Show the current automation status across all DVWDesign repositories.

This command provides a comprehensive overview of:

- Automation levels per repository

- Active cron jobs

- MCP health status

- Recent rollout history

- Pending automated tasks

- Git hook status

- DocuMind scan status

Usage: `/automation-status` or `/automation-status --detailed`

The command will analyze:

1. Repository automation metrics

2. Cron job execution logs

3. MCP server health

4. Hook installation status

5. Recent automated operations

6. Upcoming scheduled tasks

Output includes:

- Summary dashboard

- Automation percentage per repository

- Last execution times

- Error/success counts

- Recommendations for improvement
