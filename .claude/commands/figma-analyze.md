# Figma Component Analysis

Using the Figma MCP, analyze the component at the provided Figma URL or node ID.

## What to Provide

Please provide ONE of the following:
- **Option A:** Full Figma URL (e.g., `https://figma.com/file/ABC123/Design?node-id=1:2`)
- **Option B:** File key + Node ID separately

## What I'll Analyze

I will use the Figma MCP tools to:

1. **Component Structure**
   - Hierarchy and nesting
   - Auto-layout properties
   - Component variants (if any)

2. **Design Tokens**
   - Colors (fills, strokes)
   - Typography (font family, size, weight, line height)
   - Spacing (padding, margins, gaps)
   - Border radius and effects

3. **Component Properties**
   - Props and variants
   - Boolean properties
   - Instance swap properties

4. **Dev Annotations**
   - Dev comments (if any)
   - Code Connect mappings (if configured)
   - URL annotations

5. **Implementation Recommendations**
   - Suggested component structure
   - Props interface (TypeScript)
   - Styling approach (CSS/MUI/Tailwind)
   - Code example

## Output Format

I'll provide:
- ✅ Structured markdown analysis
- ✅ Design token mappings (CSS variables, MUI theme)
- ✅ React/TypeScript code suggestions
- ✅ Storybook story template (optional)
