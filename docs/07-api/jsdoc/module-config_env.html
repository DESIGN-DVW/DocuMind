config/env

Centralized environment configuration for DocuMind

This is the single source of truth for all runtime configuration. It loads .env via Node 22's built-in process.loadEnvFile() and exports all config as named constants with sensible defaults.

IMPORTANT: This module must NOT import from config/constants.mjs to avoid circular dependencies.
