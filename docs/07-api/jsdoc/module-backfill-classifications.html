backfill-classifications

Backfill classifications for all documents that have a NULL classification.

Classification uses ordered path-matching rules. First match wins. Documents not matching any rule are classified as 'uncategorized'.

Designed to be idempotent — only processes rows WHERE classification IS NULL. Accepts an open better-sqlite3 db instance (does NOT open or close the DB).
