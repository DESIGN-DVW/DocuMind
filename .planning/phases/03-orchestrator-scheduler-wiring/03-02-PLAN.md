  <action>
  Create `orchestrator.mjs` at the project root (same level as `package.json`).

  **Imports:**
  ```javascript
  import { glob } from 'fast-glob';
  import fs from 'fs/promises';
  import path from 'path';
  import { indexMarkdown } from './processors/markdown-processor.mjs';
  import { indexKeywords } from './processors/keyword-processor.mjs';
  import { buildRelationships } from './graph/relations.mjs';
  ```

  **Main export:**
  ```javascript
  export async function runScan(db, ctx, options = {}) {
    const { mode = 'incremental', repo = null } = options;
    const startMs = Date.now();
    console.log(`[orchestrator] Starting ${mode} scan${repo ? ` for ${repo}` : ''}...`);

    let result;
    switch (mode) {
      case 'incremental': result = await runIncrementalScan(db, ctx, repo, startMs); break;
      case 'full':        result = await runFullScan(db, ctx, repo, startMs); break;
      case 'deep':        result = await runDeepScan(db, ctx, startMs); break;
      default: throw new Error(`[orchestrator] Unknown scan mode: ${mode}`);
    }

    console.log(`[orchestrator] ${mode} scan complete in ${result.durationMs}ms`);
    return result;
  }
  ```

  **Helper — getRepoFiles(ctx, repo):**
  - If `repo` is specified, find the matching entry in `ctx.repoRoots` by name and glob `**/*.md` within that path
  - If `repo` is null, glob `**/*.md` across ALL `ctx.repoRoots` paths
  - Use fast-glob with `{ ignore: ['**/node_modules/**', '**/.git/**', '**/dist/**', '**/build/**'], absolute: true }`
  - Return `Array<{ filePath: string, repoName: string }>`

  **Helper — rebuildFTS(db):**
  ```javascript
  function rebuildFTS(db) {
    db.prepare("INSERT INTO documents_fts(documents_fts) VALUES('rebuild')").run();
    console.log('[orchestrator] FTS5 documents index rebuilt');
  }
  ```

  **Helper — rebuildKeywordsFTS(db):**
  ```javascript
  function rebuildKeywordsFTS(db) {
    db.prepare("INSERT INTO keywords_fts(keywords_fts) VALUES('rebuild')").run();
    console.log('[orchestrator] FTS5 keywords index rebuilt');
  }
  ```

  **runIncrementalScan(db, ctx, repo, startMs):**
  1. Load existing doc hashes: `SELECT path, content_hash, last_scanned FROM documents` into a Map keyed by path
  2. Get all repo files via `getRepoFiles(ctx, repo)`
  3. For each file: `fs.stat()` to get mtime. If not in Map OR file mtime > last_scanned, call `indexMarkdown(db, filePath, repoName, ctx)`
  4. Track counts: `{ added: 0, updated: 0, skipped: 0 }`
  5. Call `rebuildFTS(db)` once at end
  6. Return `{ mode: 'incremental', repo, documentsFound: files.length, added, updated, skipped, durationMs: Date.now() - startMs }`

  **runFullScan(db, ctx, repo, startMs):**
  1. Get all repo files via `getRepoFiles(ctx, repo)`
  2. Call `indexMarkdown(db, filePath, repoName, ctx)` for every file
  3. Track counts
  4. Call `rebuildFTS(db)` once at end
  5. Return `{ mode: 'full', repo, documentsFound, added, updated, durationMs }`

  Note: Deviation analysis is added in Plan 04. For now, log `[orchestrator] Deviation analysis: not yet implemented` as a placeholder.

  **runDeepScan(db, ctx, startMs):**
  1. Call `runFullScan(db, ctx, null, startMs)` for full indexing
  2. Keyword refresh: `SELECT id, content FROM documents` — call `indexKeywords(db, doc.id, doc.content, ctx)` for each. Track keyword count.
  3. Call `rebuildKeywordsFTS(db)`
  4. Graph rebuild: call `buildRelationships(db)`. Capture returned edge count.
  5. Call `rebuildFTS(db)` at the end
  6. Return `{ mode: 'deep', ...fullResult, keywords: keywordCount, edges: edgeCount, durationMs: Date.now() - startMs }`

  Note: Staleness and similarity detection are added in Plan 04. Log placeholders.

  **Error handling:** Wrap each scan mode body in try/catch. Log errors with `[orchestrator]` prefix. Re-throw so callers (scheduler) can record failure.
